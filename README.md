# TCP-IP-Project
먼저 java.servlet 안의 자바파일을 전부 복사하여 자바칸에 servlet파일을 만들고 자바파일을 전부 넣음(그래야 네이버 검색 API를 쓸 수 있음)
그중 SearchServlet파일의 클라우드 키를 본인 것으로 변경, 
그 후 webapp.WEB-INF안에 있는 web.xml(#으로 해서 서치등등을 지정해줘야 하는 것을 대신 해줌)
과 lib(이게 있어야 호환이 됨)에 있는 파일 또한 넣어야 함. 
SaveServlet에 DB이름과 패스워드 바꿔야 함. 
이렇게 하면 네이버 검색 API를 사용할 수 있을 겁니다.
